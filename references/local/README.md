# Local References

This directory contains local reference material used while designing or
implementing Fit Scroller.

Its contents are ignored by git and are not intended to be published with the
project repository.

Use this directory for local copies of external examples, experiments or notes
that are useful during development but should not become part of the public
project history.

## Contents

### `hyprland-integration/`

Local working notes and experiments for Hyprland `0.55` integration checks.

Use this directory to track runtime observations, unresolved API questions and
implementation decisions that depend on local Hyprland experiments.

### `hyprland-layout-examples/`

Local copies of Hyprland layout examples used to understand the custom layout
API.
